---
description: When designing a frontend or frontend components
globs: frontend/**
---

- Pay attention to visual clarity and contrast. Do not place white text on a white background.
- Ensure the UX is pleasant. Boxes should grow to fit their contents, etc.
- When asking the user for sensitive information, use password-style text-entry boxes in the UI.
- Use Next.js as it works best with Vercel.
- The frontend will ultimately be deployed on Vercel, but it should be possible to test locally.
- Always provide users with a way to run the created UI once you have created it.
